# Sports Event Question Prompt

Generate Polymarket-style sports benchmark questions from structured sports events.

Rules:
- Prefer yes/no questions.
- Use finite options only.
- Bind every question to a resolution date and public resolution rule.
- Do not ask subjective analysis questions.
- For unresolved events, keep the answer unknown and future-facing.
- For already-resolved events, rewrite the question in a pre-event style and include the known answer separately.
